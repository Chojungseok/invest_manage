# 투자 OS — 개인 투자 판단 관리 시스템

단순 자산관리 앱이 아니라 **"내가 왜 투자했고, 그 판단이 시간이 지난 뒤 맞았는지 복기하는"** 투자 판단 관리 시스템입니다.
수익률 예측이 아니라 **투자 판단의 기록 · 복기 · 점수화**가 핵심입니다.

## 실행 방법

별도 설치나 서버 없이 `index.html` 파일을 브라우저(Chrome, Edge 등)에서 열면 바로 동작합니다.

- 모든 데이터는 브라우저 **localStorage**에 저장됩니다 (외부 전송 없음)
- 처음 열면 샘플 데이터가 들어 있으며, `설정 → 데이터 초기화`에서 삭제할 수 있습니다
- `설정 → 데이터 백업`에서 JSON 백업 다운로드/복원이 가능합니다
- 차트(Chart.js)는 CDN을 사용하므로 인터넷 연결 시 표시됩니다

## 주요 기능 (1차 MVP)

| 메뉴 | 설명 |
|---|---|
| 홈 | 총자산, 올해 목표 달성률, 이번 달 투자금, 목표 월 배당금 + 자산 추이/비중 차트, 각 영역 요약 |
| 포트폴리오 | 보유 종목 등록·수정·삭제, 평가금액·평가손익·수익률·비중 자동 계산, 계좌/유형 필터, 비중 초과 경고 |
| 투자기록 | 매수/매도/추가매수/관망/리밸런싱/배당 재투자 기록. 이유·가설·리스크·감정 상태·재검토 기준 저장 |
| 판단점수 | 매수 전 판단의 질을 7개 항목 100점 만점으로 점수화 (80점 이상 양호 ~ 40점 미만 감정 매매 가능성) |
| 판단 리플레이 | 매수 기록별 1/3/6개월 복기. **수익률과 판단 점수를 분리 평가** — "수익 났지만 운이 좋았을 가능성" 판별 |
| 관심종목 | 목표 진입가·매수 조건·우선순위 관리, 목표가 도달 표시. 충동매수 방지 목적 |
| 월간계획 | 월 투자 가능 금액과 목표 비중 설정, 현재 비중 vs 목표 비중 비교 및 배분안 제안 |
| 통계 | 자산 추이, 월별 투자금, 판단 점수 추이, 감정 상태별 투자 결과, 리플레이 완료율, 수익/손실 투자의 판단점수 비교 |
| 설정 | 목표 자산, 목표 비중, 종목별 최대 비중, 환율 기준, 복기 주기, 데이터 백업/초기화 |
| 자동매매 | 준비중 페이지 (증권사 API 연동 및 모의투자 검증 이후 제공 예정) |

## 화면 구성

- 데스크톱: 좌측 사이드바 + 카드형 그리드 대시보드
- 모바일: 하단 5탭 네비게이션(홈/포트폴리오/기록/리플레이/통계), 반응형 카드 레이아웃
- 디자인: 흰색/연회색 배경, 파란색 포인트, 수익 초록/손실 빨강/경고 주황

## 데이터 모델

localStorage 키 `investOS_v1` 하나에 JSON으로 저장됩니다.

- `settings` — 목표 자산, 목표 비중, 최대 비중 한도, 환율, 복기 주기
- `assets` — 보유 종목 (계좌, 유형, 시장, 통화, 수량, 평균단가, 현재가, 환율)
- `records` — 투자기록 (행동 유형, 이유, 가설, 리스크, 감정 상태, 재검토/매도 기준)
- `scores` — 판단점수 (7개 항목 배점 + 총점 + 해석)
- `replays` — 리플레이 (1/3/6개월 일정, 가설 상태, 수익률, 원인, 다음 행동) — 매수 기록 저장 시 자동 생성
- `watchlist` — 관심종목 (목표 진입가, 매수 조건, 우선순위, 상태)
- `plans` — 월간계획 (예산, 목표 비중, 우선순위)
- `snapshots` — 월별 총자산 스냅샷 (통계 페이지에서 수동 저장)

## 1차 MVP 범위 제외 (추후 확장 예정)

- 자동매매, 증권사 API 연동, 실시간 시세, AI 종목 추천, 뉴스 자동 분석
- 주가·환율 등 모든 시세는 사용자가 직접 입력

## 기술 스택

- 순수 HTML/CSS/JavaScript 단일 파일 (프레임워크 없음)
- Chart.js 4 (CDN)
- 저장소: 브라우저 localStorage

## 주의

이 도구는 개인 기록·복기용이며 투자 권유나 종목 추천이 아닙니다. 모든 투자 판단과 책임은 사용자 본인에게 있습니다.
